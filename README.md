# T-UI Linux CLI Launcher (Modern Fork)

> **Note:** This is a **fork** of the original T-UI Launcher by Francesco Andreuzzi. The original project is outdated and no longer maintained.

## About this Fork
I really wanted to run a CLI launcher on my modern device, but the original T-UI hasn't been updated in years and is unstable on newer Android versions (Android 14+).

**My goal is simple:** Make it run and be usable on modern Android devices.

## Contributing
**Pull requests** are extremely welcome, especially given my disdain for Java. 

If you plan to make a major change, please open an Issue first to discuss it.

## How to format a bug report
1. Open an **Issue** in this repository.
2. Describe the issue, when it happens, how to reproduce it.
3. Include your Android version and Device model.
4. Include any **screenshot** or **logcat** output that you think could help.
5. Include any **file** that you think could help (*behavior.xml*, *ui.xml*, etc..).
